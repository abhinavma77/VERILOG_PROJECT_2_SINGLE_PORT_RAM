# VERILOG_PROJECT_2_SINGLE_PORT_RAM

## Project Overview

Design and verification of a Single Port RAM using Verilog HDL.

## Specifications

- Memory Depth: 16
- Data Width: 8 bits
- Address Width: 4 bits
- Synchronous Write
- Synchronous Read
